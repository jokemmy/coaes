# `coaes-cli`

> **coaes** 的命令行工具

## Usage

***coaes -v/-version***

查看版本

***coaes dev***

开启开发服务

***coaes build***

打包项目文件

***coaes start***

发布项目

***coaes test***

运行项目单元测试

***coaes init***

在当前文件夹初始化，使用模板 **coaes-sample-react**

***coaes create <projectName>***

在当前文件夹下创建文件夹并初始化项目
